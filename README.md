## 下载

### Docker

```
docker pull apachecn0/beginnersbook-zh
docker run -tid -p <port>:80 apachecn0/beginnersbook-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install beginnersbook-zh
beginnersbook-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g beginnersbook-zh
beginnersbook-zh <port>
# 访问 http://localhost:{port} 查看文档
```
